# Ferramenta de consulta DNS
Este projeto visa implementar uma ferramenta de SCAN DNS. A construção de mensagens DNS, como também a codificação e decodificação delas é feita sem o uso de métodos prontos. Este projeto inclui um cliente DNS UDP e TCP sobre TLS.

# Execução
Para executar este projeto, basta executar "App.java", seja usando `java App.java` (ou semelhantes) ou rodando através do Java Extension do VSCode. Demais instruções estão contidas na execução do programa, como escolher:

* Tipo de cliente (UDP ou TCP)
* Servidor DNS
* Domínio a ser enviado a consulta
* Quantidade de consultas a serem feitas

Informação sobre tempo de resposta e perda de pacote aparecerão somente após execução de pelo menos, uma consula.
